# Project Notes

## Overview

Aid is a landing page for verifying AI-generated claims with source-backed evidence and clear trust signals.

## Included Files

### App and Sections
- `src/main.jsx` - React entry point
- `src/App.jsx` - Main page composition
- `src/styles.css` - Global styles and design tokens
- `src/components/` - Shared atoms and page sections

### Tooling
- `vite.config.js` - Build configuration
- `package.json` - Scripts and dependencies
- `.eslintrc.cjs` - Lint rules
- `.gitignore` - Ignore rules
- `index.html` - HTML entry point

### Documentation
- `README.md` - Project overview
- `QUICK_START.md` - Fast setup guide
- `STRUCTURE.md` - Directory and component guide
- `START_HERE.md` - Short project summary

## App Features

- Hero section with the core product message
- Verification-focused content blocks
- Comparison, workflow, and integrations sections
- Testimonials, FAQ, CTA, and footer
- Tweaks panel for live design adjustments

## Commands

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Build Output

The production build creates optimized HTML, CSS, and JavaScript assets in `dist/`.

## Notes

- The app uses React 18 and Vite
- The page is styled with CSS custom properties
- Sections are split into reusable components for easier updates
