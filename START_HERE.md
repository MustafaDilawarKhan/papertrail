# Start Here

Aid is a landing page for source-backed AI claim verification. It presents the product story, feature set, verification flow, and supporting trust content in one page.

## Run It

```bash
npm install
npm run dev
```

The app opens at `http://localhost:5174` or the next open port.

## What’s Included

- A hero section with the core product message
- A trust bar and verification demo
- Feature, comparison, workflow, and integrations sections
- Testimonials, FAQ, CTA, and footer content
- A floating tweaks panel for live design adjustments

## Main Files

```
src/main.jsx         React entry point
src/App.jsx          Main page composition
src/styles.css       Global styles and design tokens
src/components/      Section components and shared atoms
src/tweaks-panel.jsx Live design controls
```

## Commands

```bash
npm run dev      # Development server
npm run build    # Production build
npm run preview  # Preview production build
npm run lint     # Lint the codebase
```

## Customization Notes

- Edit `src/styles.css` to update colors, spacing, and typography
- Edit `src/App.jsx` to change section order or content
- Edit files in `src/components/` to update individual sections
