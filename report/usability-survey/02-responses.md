## Section F — Open-ended responses (themed)

### F1. Most confusing screen

- "Library — couldn't tell at first which icon meant 'upload' vs 'new collection'." (P2, P9, P10)
- "Editor had too many pieces on first view — I didn't know where to start typing." (P4, P9)
- "Settings page looked fine but I expected a 'log out' button at the top, not the bottom." (P5)

### F2. Missing features

- **Reference import from DOI / arXiv URL** — P3, P6, P7, P8 (all heavy Zotero/Mendeley users).
- **Side-by-side compare of two PDFs.** — P3, P7.
- **In-PDF search.** — P5, P11.
- **Offline / desktop mode.** — P8, P11.
- **Voice input for prompts** while reading. — P12.

### F3. What would build trust in AI answers

- "Show me which sentences in the source were used, not just the page." (P1, P3, P6, P7)
- "Add a confidence indicator." (P4, P8)
- "Let me see the exact prompt that was sent." (P7)
- "Don't answer if no source is found — say 'not in this document'." (P3, P6, P11)

### F4. Would you use it for FYP / thesis?

- **Yes — 10/12.** Most cited the AI chat + citation cards as the killer feature.
- **Maybe — 2/12** (P9, P11). P9 cited familiarity with existing tools; P11 wanted offline mode first.

### F5. Other comments

- "Make the AI panel collapsible — sometimes I just want to read." (P5, P8)
- "Keyboard shortcuts please — Ctrl+K for command palette." (P3, P7)
- "Light mode is fine but please ship a true dark mode." (P6)
- "I love that citations are clickable. That alone would make me switch from Zotero." (P3)
