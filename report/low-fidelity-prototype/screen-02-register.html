Low-fi design notes

  • Two-step sign-up: account → verify. The form is narrow enough to feel light; the marketing column reassures while the user types.
  • Password strength meter is shown on every keystroke; “Medium” / “Strong” shown next to the bar.
  • 6-digit code split into 3-and-3 with a small dash between for readability.
  • Resend link is greyed out for 30 seconds to prevent inbox spam.
