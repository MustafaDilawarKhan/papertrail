Key design decisions captured at low-fi

  1. Two-pane document viewer. AI chat permanently on the right of the document so the user never loses context.
  2. Persistent left sidebar across every authenticated screen — same chrome on every page.
  3. Multi-document tabs above the viewer — researchers frequently jump between papers.
  4. Citation cards inline in chat — every AI answer surfaces clickable source cards that scroll to the source paragraph.
  5. One global search in the top bar — documents + citations + chat history.
  6. Editor as a separate "mode", not a sub-page of the library — writing has a different flow than reading.
